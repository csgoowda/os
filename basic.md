# 🖥️ What is Operating System?

**Definition (Interview Standard):**
An **Operating System (OS)** is **system software** that **acts as an interface between user and computer hardware**, and **manages system resources**.

**In simple words:**
OS helps users **run programs** and **use hardware easily**.

---

## ✅ What OS Does

An Operating System:

* Controls **hardware**
* Runs **programs**
* Connects **user and computer**
* Manages **CPU, Memory, Files**

---

## 💻 Examples of Operating Systems

* Windows
* Linux
* macOS
* Android
* iOS

💡 **Interview Tip:**
If asked, say:

> "Linux and Windows are the most commonly used operating systems."

---

# 🧩 Types of Operating System

These are **frequently asked in interviews**.

---

## 1️⃣ Batch Operating System

**Definition:**
Batch OS executes **multiple jobs together** without user interaction.

**Example:**
Old **mainframe computers**

**Key Point:**
User does not interact while job is running.

---

## 2️⃣ Time Sharing Operating System ⭐ (Most Important)

**Definition:**
Multiple users share CPU **using time slices**.

Each user gets **small CPU time**.

**Examples:**

* Linux
* Windows
* UNIX

**Why Important:**
Most modern systems use **time sharing**.

💡 **Interview Line:**

> "Time-sharing OS allows multiple users to use CPU efficiently by dividing CPU time into small slices."

---

## 3️⃣ Real-Time Operating System (RTOS)

**Definition:**
RTOS gives **fast response within fixed time**.

Used in **time-critical systems**.

**Examples:**

* Air traffic control
* Robotics
* Medical systems

**Key Feature:**
Very **fast and predictable** response.

---

# ⚙️ Functions of Operating System

These **5 functions are very important** for exams and interviews.

---

# 1️⃣ Process Management ⭐

**Definition:**
OS manages **running programs**, called **processes**.

It:

* Creates processes
* Schedules CPU
* Stops processes

**Example Programs:**

* Chrome
* VS Code
* Music Player

Each runs as a **separate process**.

---

# 2️⃣ Memory Management

**Definition:**
OS controls **RAM usage**.

It:

* Allocates memory
* Frees memory
* Tracks used memory

**Example:**
If you open many apps, OS decides **how much RAM each app gets**.

---

# 3️⃣ File Management

**Definition:**
OS manages **files and folders**.

It:

* Creates files
* Deletes files
* Reads/Writes files

**Example Files:**

* notes.txt
* image.jpg
* program.cpp

---

# 4️⃣ Device Management

**Definition:**
OS controls **hardware devices**.

Devices include:

* Keyboard
* Mouse
* Printer
* Monitor

Uses **device drivers**.

---

# 5️⃣ Security Management

**Definition:**
OS protects **system and data**.

It:

* Controls user login
* Sets file permissions
* Prevents unauthorized access

**Example:**
Password login screen.

---

# 🎯 Very Important Extra Topics (Add to Your Notes)

These are often asked in interviews.

---

# 🧠 Kernel

**Definition:**
Kernel is the **core part of Operating System**.

It directly interacts with:

* CPU
* Memory
* Devices

Without kernel → OS cannot run.

---

# 🧠 System Call

**Definition:**
A **system call** allows a program to **request services from OS**.

Examples:

* open file
* read file
* write file

---

# 🧠 User Mode vs Kernel Mode

| Feature | User Mode    | Kernel Mode        |
| ------- | ------------ | ------------------ |
| Access  | Limited      | Full               |
| Used by | Applications | OS                 |
| Risk    | Safe         | Dangerous if error |
